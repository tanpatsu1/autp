# Product Plan

## MVP の目的
- 商品 URL からサイズ表を取得し、ユーザーの採寸プロフィールと重ねてサイズ判断を早くする

## 対象ユーザー
- ネットで服を買う前にサイズが合うかを短時間で判断したい人
- 特にトップスとパンツのサイズ迷いが多い人

## スコープ内
- トップス / パンツのカテゴリ判定
- ルールベースのサイズ表抽出
- 不足寸法の手入力補完
- 3D 風の簡易プレビュー
- ローカル保存と Supabase 履歴保存

## スコープ外
- 商品名検索
- 高精度な物理シミュレーション
- 決済や購入導線
- 複数商品の比較画面

## 成功条件
- 主要導線が短い文脈で実装・修正できる
- UI 文言に文字化けがない
- `npm run verify` が常に通る
